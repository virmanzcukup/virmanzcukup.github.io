  CARA LOGIN

  • Pastikan area anda tercover dengan signal wifi kami

  • Akfitkan wireless pada smartphone atau laptop anda dan koneksikan dengan SSID Hotspot voucher

  • Tunggu beberapa saat sampai notifikasi pop-up muncul untuk login, jika tidak ada pop-up notifikasi di atas layar, silakan buka salah satu aplikasi (chrome, mozilla , web, browser, internet, atau opera mini). pada smartphone atau laptop anda, dan silahkan ketik manual pada halaman url browser dengan mengetik :
    http://hs.vcnet.id

  • Setelah anda berhasil masuk ke halaman login silakan aktivasi ID VOUCHER dengan mengisi username & password

  NOTIFIKASI ERROR

  • [ VOUCHER HAS ACTIVED ]
    Notifikasi ini akan tampil jika :
    Kode voucher yg anda masukan sudah pernah digunakan perangkat lain cek terlebih dahulu voucher anda voucher baru atau voucher bekas.

  • [ VOUCHER ERROR ]
    Notifikasi ini akan tampil jika :
    anda salah mengetikan ID VOUCHER, harap selalu perhatikan penulisan huruf besar dan kecil lakukan pengisian voucher manual dengan mengisi username & password

  • [ EXPIRED ]
    Notifikasi ini akan tampil jika :
    masa aktif atau waktu ID VOUCHER anda sudah berakhir.

  • [ KUOTA HABIS ]
    Notifikasi ini akan tampil jika :
    kuota ID VOUCHER anda sudah mencapai batas pemakaian.

  • [ BROWSER ERROR ]
    Notifikasi ini akan tampil jika :
    aplikasi browser pada perangkat anda tidak support login silahkan login secara manual, dengan membuka aplikasi (chrome, opera mini, web, mozila, internet, atau browser). jika tetap gagal, restart hp anda atau beli hp baru

 MASA AKTIF

  • Masa Aktif Voucher terhitung sejak pertama kali digunakan / login

  • Jika pada Vocuher anda tertera Masa Aktif maka cara perhitungannya sebagai berikut,
    contoh :
    Expired : 3 jam
    masa aktif : 3 hari
    artinya, jika voucher hanya digunakan selama 1 jam, maka 2 jam yg tersisa, masih bisa di gunakan di hari ke-2 atau hari ke-3 sejak pertama kali diaktifkan

  TIPS ❕

  • Kami menyediakan pilihan paket yang beragam voucher

  • Jika area rumah anda tercover signal kami dengan kualitas yang bagus sebaiknya pilih paket voucher mingguan / bulanan

  LAINNYA

  • Jika anda masih bingung cara berlangganan atau aktivasi voucher hotspot kami, silahkan hubungi admin layanan

  • Jika terjadi masalah seperti gangguan jaringan , dan lain-lain mohon untuk segera menginformasikannya pada kami

  • Selama anda menjadi pelanggan kami, semua masalah seputar layanan internet hotspot akan kami bantu.
